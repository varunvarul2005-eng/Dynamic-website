# Atelier E-Commerce

A premium, fully responsive e-commerce experience built with HTML, CSS, and vanilla JavaScript.

## Features
- Responsive home, products, product detail, cart, wishlist, and checkout pages
- Dynamic product loading from DummyJSON
- LocalStorage-based cart and wishlist
- Live search, filters, sorting, pagination, theme toggle, toast notifications, and scroll-to-top

## Run locally
Open index.html in a browser, or serve the folder with a simple static server.

Example:
```bash
python -m http.server 8000
```
Then visit http://localhost:8000/
